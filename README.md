# ⚔ Albion Invoice

Generador de facturas temático para **Albion Online**. Crea, personaliza y exporta facturas con estética medieval oscura para gestionar préstamos de plata, alquiler de islas, servicios de guild y cualquier concepto facturable entre jugadores o clanes.

**[→ aoinvoice.crintech.pro](https://aoinvoice.crintech.pro)**

---

## ✨ Características

- **Preview en tiempo real** — la factura se renderiza al instante según rellenas el formulario
- **Emisor y receptor configurables** — elige entre Clan, Jugador o tag personalizado
- **Categorías y conceptos dinámicos** — añade secciones ilimitadas (Préstamos, Islas, Servicios…) con sus ítems
- **Badges por ítem** — T1 · T2 · T3 · T4 · T5 · T6 · MAX TIER · o cualquier texto libre
- **Pagos y vencimientos** — múltiples fechas de pago, con opción de destacar (oro) o no
- **Total automático** — suma todos los conceptos, con etiqueta personalizable y opción de ocultarlo
- **Notas al pie** — campo libre para condiciones, observaciones o cualquier texto
- **Pie de página personalizable** — texto libre para el footer de la factura
- **Exportar a PDF** — imprime directamente desde el navegador con colores preservados (A4)
- **Bilingüe** — interfaz y factura en Español / English con un click
- **Sin backend** — 100% estático, no se guarda ningún dato

---

## 🎨 Diseño

Estética de factura medieval oscura inspirada en Albion Online:

- Fondo negro con matices cálidos
- Tipografía **Cinzel** para títulos y **EB Garamond** para cuerpo
- Oro `#c9a227`, plata `#9db8d2`, pergamino `#e8d9b0`
- Marcos de esquina, divisores con espadas ⚔, badges dorados
- Print-color-adjust para preservar el tema oscuro al exportar a PDF
